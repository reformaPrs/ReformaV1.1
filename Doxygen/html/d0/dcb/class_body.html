Подробное описание

The Body class - класс фасад всего тела.

Управляет всеми частями тела, может существовать только один экземпляр (Singleton)

enum Body::ActiveBodyPart

The ActiveBodyPart enum - варианты состояний интерактивности тела

Вехняя часть тела Нижняя часть тела Всё тело

Элементы перечислений

DOWN_BODY 

Нижняя часть тела (штаны)

Всё тело

Ничего
